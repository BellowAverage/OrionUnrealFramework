## Deprecation Announcement

Since this idea has attracted interest from other partners, I have decided not to publish it as an open-source framework at this stage. Instead, this project will become a closed-source development and serve as the foundation for a Top-Down RPG Resource Management Game, which will later be published on the Windows/Steam platform. If you are interested in participating in the development—whether in a technical or artistic capacity—please contact me at chriswang2025@u.northwestern.edu to join our team.

As a result, this repository will receive no further major commits after April 15, 2025.

## About the Framework

Main source: https://github.com/BellowAverage/OrionUnrealFramework/tree/main/Source/Orion

OrionUnrealFramework is designed to provide a better template for Top-Down game, which features a well-develped socket-grid-based building system solution and a lightweight AI action queue/tree serving as a replacement of the original blackborad-based one.
